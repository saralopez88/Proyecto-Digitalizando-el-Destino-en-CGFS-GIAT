# Proyecto-Digitalizando-el-Destino-en-CGFS-GIAT
Recurso Educativo Abierto (REA) creado con eXeLearning para el módulo de Destinos Turísticos (CFGS). Incluye guía de seguridad digital, curación de contenidos y propuesta de evaluación DUA.
Recurso Educativo Abierto (REA) - CFGS GIAT
Este repositorio contiene el material interactivo diseñado para el alumnado del Ciclo Formativo de Grado Superior en Guía, Información y Asistencias Turísticas.

Autoría: Sara López Calo.

Herramienta: eXeLearning 2.9 (o la versión que uses).
